Song Data Analysis EDA Project
Project Overview
This project involves performing Exploratory Data Analysis (EDA) on a dataset of songs. The aim is to uncover interesting insights and patterns related to various song attributes, such as genre, popularity, duration, and more. The analysis is intended to help understand trends in the music industry and provide a foundation for further predictive modeling or recommendations.

Objectives
Data Cleaning: Handle missing values, duplicates, and outliers.
Data Visualization: Use various plots and charts to visualize the distribution and relationships between different song attributes.
Feature Analysis: Analyze features like genre, artist, duration, tempo, and popularity.
Trend Analysis: Identify trends over time, such as changes in song length or popularity by year.
Correlation Analysis: Determine correlations between different features, such as tempo and danceability.
Dataset
The dataset used for this project includes attributes such as:

Song title
Artist
Genre
Duration
Tempo
Popularity
Release year
Tools and Technologies
Python: Programming language used for the analysis.
Pandas: For data manipulation and analysis.
Matplotlib and Seaborn
